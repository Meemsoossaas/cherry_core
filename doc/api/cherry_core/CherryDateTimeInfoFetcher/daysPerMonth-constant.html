Map<int, int> const daysPerMonth

A map containing the data how many days each month has. The keys are copied by the static constants of DateTime. These being as follows:

  • DateTime.january
  • DateTime.february
  • DateTime.march
  • DateTime.april
  • DateTime.may
  • DateTime.june
  • DateTime.july
  • DateTime.august
  • DateTime.september
  • DateTime.october
  • DateTime.november
  • DateTime.december
