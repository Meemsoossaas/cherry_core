isActivated property

bool isActivated

Gets a bool if this is activated. On default, it gets true. If it gets true then this is fully functional. If it gets false then this is fully disabled. To change this, just set it to the opposite bool value.

print(service.isActivated) // Outputs: true
service.isActivated = false; // 'service' is now deactivated.

void isActivated=(bool isActivated)
